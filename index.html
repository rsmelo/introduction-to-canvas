Simple Shapes

Syntax Action
fillRect(x, y, width, height) draws a filled rectangle
strokeRect(x, y, width, height) draws the outline of a rectangle
clearRect(x, y, width, height) sets all pixels in the rectangle to transparent black

Paths

  • A path is a list of subpaths
  • subpaths can be of different shapes, curved or not

Basic Path Methods

Syntax Action
beginPath() starts a new path
moveTo(x, y) moves the starting point of a new sub-path
closePath() causes the point of the pen to move back to the start of the current sub-path
fill() fills the sub-paths
stroke() outlines the sub-paths

Basic Path Methods

Syntax Action
lineTo(x, y) draws a straight line from the previous point to x,y
arc(x, y, radius, startAngle, endAngle, anticlockwise) adds an arc to the path
arcTo(x1, y1, x2, y2, radius) adds an arc to the path connected to the previous point by a straight line
bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y) adds a cubic Bézier curve to the path
quadraticCurveTo(cpx, cpy, x, y) adds a quadratic Bézier curve to the path

//ctx.lineTo(x, y);
